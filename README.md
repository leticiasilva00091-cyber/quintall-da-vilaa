# 🍽️ Cardápio Digital — Quintal da Vila

Site do cardápio digital do **Quintal da Vila Restaurante**, desenvolvido em HTML, CSS e JavaScript puro. Funciona sem nenhuma instalação — basta abrir o `index.html` no navegador ou publicar no GitHub Pages.

## ✅ Funcionalidades

- Cardápio completo com todas as categorias e preços
- Filtro por categoria (Executivo, Porções, Réchaud, Parmegiana, Hambúrguer, Buffet, Cervejas, Sem Álcool)
- Fotos realistas de cada categoria
- Carrinho de compras com controles de quantidade
- Total calculado automaticamente
- Envio do pedido direto pelo WhatsApp
- Drawer "Sobre Nós" com informações do restaurante
- Design responsivo (mobile e desktop)
